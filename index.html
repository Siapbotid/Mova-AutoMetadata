Quick Start Tutorial

1

Configure API

Select your AI platform (OpenAI or Gemini) and enter your API key. You can get keys from their respective websites.

2

Select Media

Choose individual files or select an entire folder containing images, videos, or vector files (SVG, EPS).

3

Adjust Settings

Set your preferred title length, keyword count, and choose output folders for processed files.

4

Generate Metadata

Click "Generate Metadata" and watch as AI analyzes your media and creates titles, keywords, and descriptions.

5

Review Results

Check the results panel to see processed files. Metadata is embedded in files and exported to CSV for easy management.

Pro Tips:

  • Use multiple API keys for faster processing
  • Gemini is recommended for better cost efficiency
  • SVG files work better with Gemini than OpenAI
  • Check the CSV files for easy metadata management
